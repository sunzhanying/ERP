# 进销存ERP系统

# 技术框架
* 核心框架：Spring Framework 3.0.5
* 视图框架：Struts 2.2.3
* 持久层框架：Hibernate 3.0
* 日志管理：Log4j 1.2.16
* JS框架：Jquery 1.8.0
* UI框架: EasyUI 1.3.5
* 桌面框架: HoorayOS
* 项目管理框架: Maven3.2.3

# 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: MyEclipse8.5+/Eclipse4.4+
* DB: Mysql5.1
* JDK: JDK1.6+
* WEB: Tomcat6.0+
* Maven: Maven3.2.3+

# 运行环境
* WEB服务器：Tomcat6.0+
* 数据库服务器：Mysql5.1
* JAVA平台: JRE1.6+
* 操作系统：Windows、Linux
